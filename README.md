# 🌿 BOTANICA - AI Medicinal Plant Identification System

## 📌 Project Description

BOTANICA is a deep learning-based web application that identifies medicinal plants from images using a Convolutional Neural Network (CNN) model. The system is built using **Flask (Python)** for the backend and utilizes **DenseNet121** with transfer learning for accurate classification.

This project helps in preserving traditional knowledge and makes medicinal plant identification accessible to everyone.

---

## 🚀 Features

* 🌱 Upload plant images (leaf / stem / full plant)
* 🤖 AI-based classification using CNN (DenseNet121)
* ⚡ Fast and accurate predictions
* 🌐 User-friendly Flask web interface
* 📊 Scalable deep learning model

---

## 🧠 Tech Stack

* **Frontend**: HTML, CSS (Flask Templates)
* **Backend**: Flask (Python)
* **Model**: CNN (DenseNet121 - Transfer Learning)
* **Libraries**: TensorFlow / Keras, NumPy, OpenCV

## 🧪 Model Details

* Architecture: **DenseNet121**
* Type: Transfer Learning
* Optimizer: Adam
* Loss Function: Categorical Crossentropy
* Pooling: Global Average Pooling
* Regularization: Dropout
